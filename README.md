# Web Development Capstone

A capstone project demonstrating AI-assisted development practices, from environment setup to a working toolchain using Claude Code and Git.

## Tech Stack
- HTML5, CSS3
- Node.js (LTS v24.18.0)
- Git & GitHub

## Getting Started
1. Clone this repository
2. Ensure Node.js (LTS) and Git are installed
3. Review `CLAUDE.md` for project conventions

## Project Structure
- `README.md` – project overview
- `LICENSE` – MIT license
- `CLAUDE.md` – AI assistant conventions and stack details
- `.gitignore` – Node.js ignore rules
